<h1> Hướng dẫn chạy project: </h1>

* clone project vào máy
* mở cmd -> cd đến thư mục tiktokUI
* chạy câu lệnh trên cmd: npx expo@54 install
* chạy câu lệnh trên cmd: npx expo start
* mở expo app trên điện thoại hoặc giả lập
* ( nếu dùng điện thoại: quét mã QR code trên terminal )
* ấn phím 'a' để mở app trên android giả lập
* ấn phím 'i' để mở app trên ios giả lập
